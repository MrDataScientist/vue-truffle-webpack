# Vue + Truffle + Webpack

## Install

```
curl -O https://cdn.rawgit.com/wilfreddenton/vue-truffle-webpack/d81c8c33/vtw ; chmod +x vtw ; mv vtw /usr/local/bin
```

## Usage

1. `mkdir dapp`
2. `cd dapp`
3. `vtw`
4. Answer the prompts to setup the Vue app
5. `cd app`
6. `npm install`
7. `npm run dev`
